# Overthinking → Action Converter

A simple decision-support web app that turns messy thoughts into clear, actionable steps.

## 💡 Overview

This tool helps users break out of overthinking by converting unstructured thoughts into:

- A clear problem summary  
- 3 simple, actionable steps  
- An “overthinking score” based on input patterns  

It’s designed to simulate a lightweight “AI-style” thinking assistant using pure JavaScript logic.

---

## 🚀 Features

- ✍️ Input free-form thoughts or problems  
- 🧠 Automatic summary generation  
- ✅ 3 actionable steps based on context  
- 📊 Overthinking score (0–100)  
- 💾 Save entries using local storage  
- 📂 View and manage past entries  
- 🧹 Clear input and reset results  

---

## 🛠️ Built With

- HTML5  
- CSS3 (modern responsive UI)  
- JavaScript (vanilla, no frameworks)  
- Local Storage API  

---

## 📸 How It Works

1. Type your thoughts into the input box  
2. Click **"Convert to Action"**  
3. View:
   - Summary  
   - Action steps  
   - Overthinking score  
4. Save entries to track progress over time  

---

## 🧠 Logic Behind the App

- Detects keywords (e.g., stress, confusion, worry)  
- Measures repetition and length  
- Calculates an “overthinking score”  
- Generates simple actions based on context
